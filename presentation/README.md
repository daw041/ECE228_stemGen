# Presentation Files

This folder contains the 3-minute highlight presentation materials.

- `highlight_slides.md`: compact 4-slide content for the talk
- `speaker_script_cn.md`: Chinese speaker script matched to the slides
- `speaker_script_en.md`: English speaker script matched to the slides
- `stemgen_highlight_3min.pptx`: generated 4-slide PowerPoint deck
- slide 3 includes the full mel-spectrogram diagnostic figure for qualitative effectiveness discussion

The slides are intentionally short enough for a 3-minute oral presentation.
